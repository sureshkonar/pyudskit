# CLI

The CLI exposes quick encode/decode and AI commands.

```python
pyudskit encode "Read the VIN"
pyudskit decode "7F 22 31"
pyudskit service 0x27
pyudskit dtc P0301
pyudskit ai encode "Read the VIN"
pyudskit profile validate pyudskit/profiles/oem_example.json
pyudskit profile show pyudskit/profiles/oem_example.json
```
